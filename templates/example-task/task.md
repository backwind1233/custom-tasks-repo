**Prompt:**

This is a template task that demonstrates the expected format for migration tasks.

## Prerequisites

- Java 11 or higher
- Maven or Gradle project
- Access to target service

## Migration Steps

1. Update dependencies in pom.xml or build.gradle
2. Update configuration files with new settings
3. Transform code to use new API patterns
4. Update error handling for new exceptions

## Verification

1. Run unit tests: `mvn test`
2. Verify application starts successfully
3. Test key functionality manually
